    Question¶

    What is the seven-element template that structures each letter to the seven churches (address, self-identification, commendation, rebuke, exhortation, promise, ear formula)? Document the template elements in each letter. What does the ear/overcomer reversal between the first three and last four letters reveal — in letters 1-3 the ear formula comes before the overcomer promise; in letters 4-7 the overcomer promise comes before the ear formula? What does this 3+4 structural division signify? Does it correspond to any other 3+4 or 4+3 patterns in Revelation?

    Prior Research Summary¶

    Six prior studies directly relevant: revs-08 mapped the 8-element template with 3 invariable + 3 variable elements (SP021 Strong); revs-09 confirmed ear/overcomer reversal (SP022 Strong) and documented the N+M subdivision across septenary series (SP025 Moderate); revs-10 tested chiastic pairing with B-B' (Smyrna-Philadelphia) as Strong; revs-12 mapped overcomer-to-fulfillment arcs (8 VPs to Rev 19-22); revs2-01 documented Spirit/churches formula confinement and ekklesia distribution; revs2-05 mapped the nikao thread (17 occurrences, promise-to-fulfillment arc). Web research provided genre analysis (Aune's prophetic edict, Shea's covenant treaty, Beale's seven-part template), ear formula scholarship (Enroth's Weckformel), chiastic models (Beale A-B-C, Stefanovic/Paulien menorah 3+1+3), and 4+3 pattern documentation across all four septenary series.

    Focus Areas¶

    1. The seven-element template: mapping all elements in all seven letters: This is the spine of the entire study. The research agent must read all seven letters in full (Rev 2:1 through Rev 3:22 from KJV) and map each template element with precise verse boundaries. READ $WORD_STUDIES/G3004-lego.md for the lego/tade legei formula. READ $WORD_STUDIES/G1125-grapho.md for the 'write' command. READ $WORD_STUDIES/G1492-oida.md for the 'I know' assessment verb.
