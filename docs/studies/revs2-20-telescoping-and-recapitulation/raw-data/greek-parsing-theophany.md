# Greek Parsing Theophany

```
======================================================================
  GREEK PARSING — THEOPHANY FORMULA PASSAGES
======================================================================

=== Rev 4:5 ===
Greek: καὶ ἐκ τοῦ θρόνου ἐκπορεύονται ἀστραπαὶ καὶ φωναὶ καὶ βρονταί·
Elements: astrapai (G796, N-NPF) + phōnai (G5456, N-NPF) + brontai (G1027, N-NPF) = 3 elements
Key verb: ἐκπορεύονται (G1607) = Pres M/P Ind 3P — CONTINUOUS PRESENT (ongoing emanation, not punctiliar event)

=== Rev 8:5 ===
Greek: καὶ ἐγένοντο βρονταὶ καὶ φωναὶ καὶ ἀστραπαὶ καὶ σεισμός.
Elements: brontai (G1027) + phōnai (G5456) + astrapai (G796) + seismos (G4578, N-NSM) = 4 elements
Key verb: ἐγένοντο (G1096) = Aor Mid Ind 3P — punctiliar aorist
Note: Order reversal from 4:5 (brontai leads here); seismos added as 4th element

=== Rev 11:19 ===
Greek: καὶ ἐγένοντο ἀστραπαὶ καὶ φωναὶ καὶ βρονταὶ καὶ σεισμὸς καὶ χάλαζα μεγάλη.
Elements: astrapai (G796) + phōnai (G5456) + brontai (G1027) + seismos (G4578) + chalaza megalē (G5464 + G3173) = 5 elements
Key verbs: ēnoigē (G455) = Aor Pass Ind 3S — temple OPENED passively (divine passive)
           ōphthē (G3708) = Aor Pass Ind 3S — ark WAS SEEN (divine passive)
           ēnoigē note: the N1904 text has ēnoigē; both openings (naos and ark) are divine passives

=== Rev 16:18 ===
Greek: καὶ ἐγένοντο ἀστραπαὶ καὶ φωναὶ καὶ βρονταί, καὶ σεισμὸς ἐγένετο μέγας, οἷος οὐκ ἐγένετο ἀφ' οὗ ἄνθρωπος ἐγένετο ἐπὶ τῆς γῆς, τηλικοῦτος σεισμὸς οὕτω μέγας.
Elements: astrapai (G796) + phōnai (G5456) + brontai (G1027) + seismos megas (G4578 + G3173) = 3+1(intensified)
Superlative: seismos megas hoios ouk egeneto — "such as was not since men were upon the earth"
Note: chalaza returns at v.21 — "great hail out of heaven, every stone about the weight of a talent"
Total elements at 16:18-21: all 5 elements present + superlative quantification
```
