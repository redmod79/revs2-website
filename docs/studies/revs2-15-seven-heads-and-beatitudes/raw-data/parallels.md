# Cross-Testament Parallels

- Rev 17:9 → Rev 12:3 (nt, vocabulary (hepta + kephalai))
- Rev 17:9 → Rev 13:1 (nt, vocabulary (hepta + kephalai + deka kerata))
- Rev 17:9 → Rev 5:6 (nt, vocabulary (hepta — seven horns/eyes of Lamb))
- Rev 17:9 → Dan 2:35 (ot, thematic (mountain = kingdom symbolism))
- Rev 17:9 → Jer 51:25 (ot, thematic (mountain = empire — Babylon as destroying mountain))
- Rev 1:3 → Rev 22:7 (nt, vocabulary + structural (inclusio — makarios + tereo + logous + propheteias))
- Rev 1:3 → Luk 11:28 (nt, thematic (blessed are they that hear and keep the word))
- Rev 17:11 → Rev 17:8 (nt, vocabulary + structural (beast temporal formula: was, is not, will come/is eighth))
- Rev 17:11 → Mat 12:45 (nt, thematic (seven spirits return, last state worse — seven-then-worse pattern))
- Rev 13:3 → Rev 5:6 (nt, vocabulary (sphazo + hos — slain construction, beast parodies Lamb))
- Rev 13:3 → Rev 13:12 (nt, vocabulary (deadly wound healed — second beast enforces worship of the healed first beast))
- Ps 74:14 → Rev 12:3 (ot, thematic (multi-headed sea creature — Leviathan heads → dragon heads))
- Ps 74:14 → Isa 27:1 (ot, vocabulary + thematic (Leviathan as serpent/dragon in the sea))
- Rev 14:13 → Rev 19:9 (nt, structural (Grapson imperative — only beatitudes 2 and 4))
- Rev 16:15 → Rev 22:7 (nt, structural (Idou erchomai formula — only beatitudes 3 and 6))
- Rev 14:13 → Rev 20:6 (nt, thematic (death/resurrection pair — die in the Lord / first resurrection))
- Rev 17:8 → Rev 1:4 (nt, structural (beast temporal formula inverts divine existence formula))
- Isa 21:9 → Rev 14:8 (ot, quotation (doubled naphal/epesan — 'Babylon is fallen, is fallen'))
