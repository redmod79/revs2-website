# Cross-Testament Parallels

- Rev 2:7 → Gen 2:9 (ot, vocabulary + structural — tree of life, midst, paradise/garden. AN013 Strong.)
- Rev 2:27 → Ps 2:9 (ot, vocabulary + structural — rod of iron, nations, potter's vessel. 3+ shared vocab. AN-tier Strong.)
- Rev 3:7 → Isa 22:22 (ot, near-verbatim quotation — key of David, open/shut, none shuts/opens. 5+ shared elements. AN012 Strong.)
- Rev 3:21 → Ps 110:1 (ot, vocabulary + thematic — sit, throne, right hand. Throne-sharing concept.)
- Rev 13:9 → Rev 2:7 (nt, modified formula — shared ous + akousato but missing ho echon participial form, missing Spirit/churches clause. TM043.)
- Rev 21:7 → Rev 2:7 (nt, vocabulary — ho nikon (present active participle same form). Universal gathering of all seven overcomer promises.)
